Page À Propos

Ce projet démontre la mise en place d'une chaîne CI/CD complète :

  1. Le code source est hébergé sur **GitHub**.
  2. Un environnement de serveur est créé par **Vagrant** et **VirtualBox**.
  3. **Jenkins** est configuré pour récupérer, construire (Build) et déployer le code sur le serveur **Apache**.
